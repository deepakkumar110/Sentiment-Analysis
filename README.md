# Sentiment Analysis on Movie Reviews

This project predicts whether a movie review is Positive or Negative using NLP and Deep Learning (LSTM).

## Technologies Used
- Python
- Pandas
- TensorFlow / Keras
- NLP
- Streamlit

## Features
- Predict Positive or Negative Review
- Deep Learning Model using LSTM
- Live Web App with Streamlit

## Project Structure
dataset/
models/
notebook/
screenshots/
app.py
requirements.txt

## Example Inputs

Positive:
This movie was amazing and I loved it

Negative:
This was the worst movie I have ever seen
